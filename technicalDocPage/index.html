What is Python

Python is an easy to learn, powerful programming language that has efficient high-level data structures and a simple but effective approach to object-oriented programming. Python is an ideal language for screipting and rapid application development in many areas of most platforms due to it´s elegant syntax and dynamic typing. The Python interpreter is easily extended with new functions and data types implemented in C or C++ (or other languages callable from C). Python is also suitable as an extension language for customizable applications.
Python is used in the development of different areas of computer science, such as:

  • AI and machine learning
  • Data analytics
  • Data visualisation
  • Programming applications
  • Web development
  • Game development
Before you start

There are some things you should be familiar with in order to take advantage of Python´s full potential in any of your projects:

  • Understanding of basic programming concepts.
  • Some programming experience. If your new to programming, please try one of Python´s official tutorials before reading the rest of the article.
First script

When programming, you can print out messages on your console log to show any piece of data you want at any moment of the execution of the code. In Python, in order to do that, you need to call the print method.
Once you have created a new Python file in your Pycharm folder, write the following text:

print("Hellow World")

This line of code should print out the message written in braquets. In order to do so, click the green arrow button on the top side of your monitor or use SHIFT + F10.

Variables

When programming in Python, we use structures called variables to store data during the execution of the program. This structures can store all sort of different things: the result of the sum of two numbers, the name of your pet, the length of the car you want are trying to build. ect... Thanks to this structures, we can run programs and create useful applications.

Using variables

In other programming languages such as C, C++ or JavaScript, when creating a new variable, you need to do two different operations in order to be able to use this new variable:

  1. Declare the variable Java var x; C int a;
  2. Give values to the variable Java x = "Eyy"; C a = 10;

In Python there is no need to do any of that. When creating a new variable, the only thing that is required to do is to use a correct identifier when naming the new variable. You don't need to take into consideration the type of the value you want to store at any moment. Beacuse of this, Python is considered an easy to learn programming language for people who want to get into the incredible world of computer science.

Data types

Although there is no need to declare the variables and specify their type when using them, there are still different types of values that variables can store. Here we list some of the more commonly used types:

  • Primitive values

    • Integers
    • Floats
    • Boolean
    • Strings

  • Objects

One cool feature of Python is the fact that when a variable is given an integer value, later on at any point in the code, it can be given a float, boolean or string value. Variables aren't tied or linked to a single type of data, and its value can be changed to any other primitive value or Object whenever it is needed.

Loops

Loops let programmers repeat certain operations numerous times without having to write the same lines 10, 20 or 1000 times. This way, programmers can avoid creating incredibly long scripts where the mayority of the lines do the same operations. In Python, this iteration of operations are often acomplished by using the while statement

while (condition):

The condition must be a direct boolean value (either True/False or a variable with any of those values) or the result of 1 or more boolean expresions.

if...else statements

When a programmer wants to express a condition that determines the posibility to choose between two options, the if and else statements are used. There are two different types of statements:

  • if...then 
    if (condition):
  • if...then else... 
    if (condition):
    code
else:
    code
Indentation

In other programming languages mention before like C or JavaScript, in order to specify which lines of code belong to a function or are considered to be inside a while loop, {} are used. That way, every line of code inside those symbols are considered part of the function or while loop that is being executed.
In Python, on the other hand, this symbols are used to create and define structures called dictionaries. In order to delimiter the lines of code that belong to a specificfunction, programmers need to make use of the indentation propertie of the text. Here are two examples of how to indent correctly and an example that shows how indentation should not be done:

Indentation in a function 
def my_function():
    print("Hi")
    x = 2
    y = 5
 Indentation in an if...else statement 
if True:
    print("Yes")
else:
    print("No")
 Incorrect indentation of a while loop 
while True:
x = 1
y = 2
z = 3
